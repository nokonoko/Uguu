Store any filetype with a size up to 150MB for up to 1 hour.

Uguu cares about your privacy and stores NO logs.

If you would like to upload using ShareX read this.

To upload using curl or make a tool you can post using:
curl -i -F name=test.jpg -F file=@localfile.jpg http://uguu.se/api.php?d=upload

If you need to store files for a longer time, please use Pomf.se or wait until Uguu offers longer time.

Uguu is in preview phase, so design is shitty and storage time is low. You can contact me via neku@pomf.se or @nekunekus.
